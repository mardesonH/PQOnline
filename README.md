# **Physiotherapy Questionnaires Online**

<img src="https://raw.githubusercontent.com/mardesonH/PQOnline/2.0/assets/img/Logo.png" alt="Logo PQ Online">

Site *open source* desenvolvido por **Mardeson Herculano** para o uso de questionários de fisioterapia online, para a Liga de Fisioterapia Esportiva - Universidade Federal do Ceará.

O **Physiotherapy Questionnaires Online** foi construído em Bootstrap e utiliza Javascript para calcular os scores de cada questionário.

<a href="https://mardesonh.github.io/PQOnline/">Teste aqui!</a>

## Features
- Questionários de diversas partes do corpo;
- Opção de imprimir os resultados (score + perguntas);
- 100% Responsivo;
- 100% Grátis

## Features futuras
- Armazenamento dos resultados internamente;
- Compartilhamento dos resultados;
- Funcionalidades Offline
